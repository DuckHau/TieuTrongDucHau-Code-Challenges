# Problem 4: Three ways to sum to n

This project contains 3 unique implementations (loop, recursion, math) for a sum function (1...n) in TypeScript.

## Implementations

1.  **Method A (Loop):** An iterative solution using a `for` loop. (O(n) Time, O(1) Space)
2.  **Method B (Recursive):** A recursive solution. (O(n) Time, O(n) Space)
3.  **Method C (Mathematical):** An optimal solution using the arithmetic series formula. (O(1) Time, O(1) Space)

## How to Run

# From this directory (problem4-Sumton)
npx ts-node src/index.ts
